# Бэкенд сервиса для проведения квизов

REST API сервис для проведения командных викторин (квизов) на время. Проект реализует полный цикл работы: регистрацию игроков, создание команд, управление вопросами и играми, а также проведение игры в реальном времени с 30-секундным таймером.

## Стек технологий
* **Python 3.11**
* **FastAPI** — асинхронный веб-фреймворк
* **PostgreSQL + asyncpg** — основная база данных
* **SQLAlchemy 2.0 + Alembic** — ORM и миграции
* **Redis** — для системы Pub/Sub и управления сессиями игр
* **Pytest** — для функционального тестирования API
* **Docker & Docker Compose** — для контейнеризации и развертывания

## Запуск проекта

Убедитесь, что у вас установлены Docker и Docker Compose.

1. Запустите сборку и старт контейнеров:
```bash
docker-compose up -d --build
2. Сервис будет доступен по адресу http://localhost:8000

3. Интерактивная документация (Swagger) доступна по адресу http://localhost:8000/docs

Запуск тестов
Тестирование выполняется внутри контейнера app. Для запуска тестов выполните команду:

Bash
docker-compose exec app pytest
